# Frontend

React and Vite frontend for FraudShield AI.

## Current Phase

Basic frontend shell with Tailwind CSS configured. No backend connection,
authentication, or ML logic has been added yet.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` previews the production build.

## Structure

- `src/components` for reusable UI components.
- `src/pages` for route-level pages.
- `src/services` for API service functions.
- `src/routes` for route configuration.
- `src/data` for mock data during early development.
